# Bolt12 Offer Decoding Library

Python binding for the [Bolt12 Offer Decoding Library](https://github.com/21M4TW/b12od). The library can be built and tested using:

```
python3 -m venv test
. test/bin/activate
pip3 install pytest
pip install .
pytest
```

Sample code is also located in [examples](https://github.com/21M4TW/b12od/tree/main/examples).
